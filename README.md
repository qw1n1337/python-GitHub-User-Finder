# GitHub User Finder

**Автор:** Архипова Ангелина

## Описание

GitHub User Finder — это графическое приложение для поиска пользователей GitHub, добавления их в избранное и сохранения списка избранных пользователей в JSON-файл.

### Возможности

- Поиск пользователей GitHub по имени или логину
- Отображение списка найденных пользователей
- Добавление пользователей в избранное
- Просмотр избранных пользователей
- Удаление пользователей из избранного
- Сохранение избранного в JSON-файл
- Копирование логина пользователя в буфер обмена
- Контекстное меню для удобного управления

## Использование GitHub API

Приложение использует GitHub REST API v3 для получения информации о пользователях.
